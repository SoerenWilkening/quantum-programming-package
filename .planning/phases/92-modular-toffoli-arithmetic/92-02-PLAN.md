**CQ modular multiplication (toffoli_mod_mul_cq): result = value * c mod N**
Algorithm (from RESEARCH.md Pattern 3):
- value is quantum (n bits, in [0, N-1]), c is classical, N is classical
- result register must be pre-allocated and initialized to |0> (n bits)
- Precompute: a_j = (c * 2^j) mod N for each j from 0 to n-1 (all classical values)
- For each bit j of value (quantum register):
    - If a_j != 0: toffoli_cmod_add_cq(result_qubits, n, a_j, N, value_qubits[j])
      (controlled modular addition of classical a_j, controlled by value[j])
- After loop: result = (value * c) mod N

Edge cases:
- c == 0 or c % N == 0: result stays |0> (no-op)
- c == 1: result = value (just copy via CX gates from value to result)

Parameters: circ, value_qubits, value_bits, result_qubits, result_bits, multiplier (int64_t), modulus (int64_t).

**Controlled CQ modular multiplication (toffoli_cmod_mul_cq):**
Same loop but each toffoli_cmod_add_cq must be controlled by BOTH value[j] AND ext_ctrl. Use AND-ancilla pattern: CCX(and_anc, value[j], ext_ctrl); toffoli_cmod_add_cq(result, n, a_j, N, and_anc); CCX(and_anc, value[j], ext_ctrl). Allocate and_anc once before the loop, free after.
