<objective>
Fix opt=1 compile behavior so replay (cache hit) skips `inject_remapped_gates()` and only records a DAG node. This prevents OOM from flat gate expansion when compiled functions are replayed many times (e.g., in the chess engine). Capture (first call) still injects gates normally.

Purpose: Required for CHESS-02 (engine compiles without OOM) and CHESS-05 (circuit stats from DAG).
Output: Modified compile.py with DAG-only opt=1 replay, plus new test file verifying the behavior.
</objective>
