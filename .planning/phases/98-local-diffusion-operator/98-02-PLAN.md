```python
def _simulate_statevector(qasm_str):
    """Run QASM through Qiskit Aer and return statevector."""
    circuit = qiskit.qasm3.loads(qasm_str)
    circuit.save_statevector()
    sim = AerSimulator(method="statevector", max_parallel_threads=4)
    result = sim.run(transpile(circuit, sim)).result()
    return result.get_statevector()
```

**CRITICAL: Test state preparation strategy.**
QWalkTree constructor prepares the root state (h[max_depth] = |1>). To test D_x at a non-root depth, we need the node to be at that depth. The simplest approach: construct the tree, then manually set the state to depth=k by:
1. Flipping the root height qubit off: emit_x on h[max_depth]
2. Flipping the target height qubit on: emit_x on h[depth]
This puts the system in state |h[depth]=1, all branches=0>.

Then apply local_diffusion(depth=k). The D_x reflection about |psi_x> should transform |h[depth]=1, branches=0> into a superposition. Since |h[depth]=1, branches=0> represents the node with parent above (at depth+1) and children below, after D_x the state should have amplitudes on:
- Parent: h[depth+1]=|1>, branch at parent's level in some state (but wait -- this needs more careful analysis)

Actually, re-read the research: |psi_x> = (1/sqrt(d+1)) * (|parent> + sum_children |child_i>). The parent state is h[depth] staying at h[depth] (parent direction involves h[depth-1] for going UP the tree... no, parent means going toward root, children mean going toward leaves).

Key insight from RESEARCH.md Pattern 2: The parent-child Ry rotation acts on h[depth-1] qubit. For a node at depth k:
- "Parent" means going UP (toward root) = h[k] stays |1>, no height transition
- "Children" means going DOWN = h[k] -> h[k-1], with branch register encoding which child

So U (state preparation for |psi_x>) starting from a reference state:
- Ry(phi) on h[depth-1]: cos(phi/2)|parent> + sin(phi/2)|children>
- Then Ry cascade on branch register to distribute among d children

After D_x on a node at depth=k starting from the state |h[k]=1, branch=0> (which is the parent component of |psi_x>), the reflection should produce:
- Amplitude on |h[k]=1, branch=0> = 2*(1/sqrt(d+1))^2 - 1 = (2/(d+1)) - 1 = (1-d)/(d+1)
- Plus amplitudes on child states

This is getting complex. A simpler test approach: verify D_x^2 = I (reflection property) and verify that D_x|psi_x> = |psi_x> (eigenstate with eigenvalue +1).
