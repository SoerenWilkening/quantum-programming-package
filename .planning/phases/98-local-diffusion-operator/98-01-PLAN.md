  <action>
Add `_setup_diffusion()` private method to QWalkTree, called from `__init__` AFTER register allocation and root state prep but BEFORE predicate validation. This method:

1. Precomputes all diffusion angles eagerly and stores in `self._diffusion_data` (list indexed by level_idx 0..max_depth-1):
   - For each level_idx in range(max_depth):
     - d = self.branching[level_idx]
     - depth = self.max_depth - level_idx
     - phi = 2.0 * math.atan(math.sqrt(d)) -- parent/children split angle
     - cascade_angles = [2.0 * math.atan(math.sqrt(1.0 / (d - 1 - k))) for k in range(d - 1)] -- d-way equal superposition
     - Store as dict: {'phi': phi, 'cascade': cascade_angles, 'd': d, 'depth': depth, 'level_idx': level_idx}

2. Precomputes root angle and stores in `self._root_phi`:
   - d_root = self.branching[0] (root is at level_idx=0)
   - n = self.max_depth
   - self._root_phi = 2.0 * math.atan(math.sqrt(n * d_root))

3. Add `_setup_diffusion()` call to `__init__` right after the root state preparation block (after `emit_x(root_qubit)`) and before the predicate validation block.

4. Add `diffusion_info(self, depth)` public method:
   - Validates depth: raises ValueError if depth < 0 or depth > self.max_depth
   - Returns a dict with keys: 'phi', 'cascade_angles', 'd', 'depth', 'is_root', 'phi_root' (only if is_root)
   - For depth=0 (leaf): returns {'depth': 0, 'd': 0, 'is_leaf': True}
   - For depth=max_depth (root): includes 'phi_root': self._root_phi and 'is_root': True
   - For other depths: includes 'is_root': False

5. Add `_height_qubit(self, depth)` private helper:
   - Returns the physical qubit index for height register bit at given depth

<task type="auto">
  <name>Task 2: Implement local_diffusion method with height-controlled dispatch</name>
  <files>src/quantum_language/walk.py</files>
  <action>
Add `local_diffusion(self, depth)` public method and `_ry_cascade()` private helper to QWalkTree. The implementation follows the U-S0-U_dagger reflection pattern per CONTEXT.md locked decisions.

**local_diffusion(self, depth):**

1. **Depth validation:** Raise ValueError if depth < 0 or depth > self.max_depth with message f"depth must be 0..{self.max_depth}, got {depth}".

2. **Leaf skip:** If depth == 0, return immediately (no gates -- leaves have no children).

3. **Get level data:**
   - level_idx = self.max_depth - depth
   - data = self._diffusion_data[level_idx]
   - d = data['d']
   - branch_reg = self.branch_registers[level_idx]

4. **Height-controlled dispatch:** Create qbool wrapper for h[depth]:
   ```python
   from .qbool import qbool
   h_qubit_idx = self._height_qubit(depth)
   h_control = qbool(create_new=False, bit_list=[h_qubit_idx])
   ```
   All subsequent gates go inside `with h_control:` block.

5. **Inside the controlled block, implement the reflection D_x = U * S_0 * U_dagger:**

   **Step A: U_dagger (inverse state preparation)**
   The state preparation U maps |0...0> to |psi_x>. U involves:
   - Parent-child split: Ry(phi) on the parent-child amplitude qubit
   - Child cascade: Ry cascade on branch register qubits

   U_dagger reverses this:
   - Apply inverse cascade: Ry(-cascade_angles[k]) in reverse order, with same control structure
   - Apply inverse parent-child split: Ry(-phi) on the height transition

   For the parent-child split:
   - If depth == self.max_depth (root): use Ry(-self._root_phi) on the branch register's first qubit (no height transition -- root has no parent)
   - Else (non-root): use Ry(-data['phi']) on h[depth-1] qubit, controlled on (nothing extra beyond the outer h[depth] control)

   The parent-child Ry rotation acts between the parent and children subspace. For non-root nodes:
   - The rotation is on the height qubit h[depth-1] with angle phi, splitting amplitude between staying at h[depth] (parent) and transitioning to h[depth-1] (children)

   Then the cascade distributes children amplitude across branch register states.

   So U_dagger = inverse_cascade * Ry(-phi on h[depth-1]):
   - First undo cascade: call `self._ry_cascade_inverse(branch_reg, d, data['cascade'])`
   - Then undo parent split: emit_ry(self._height_qubit(depth - 1), -phi) for non-root
   - For root: first undo cascade, then emit_ry on first branch qubit with -root_phi (root has no height transition to parent)

   **Step B: S_0 reflection**
   Apply S_0 (reflection about |0...0>) on the relevant qubits.
   - For non-root: the local subspace involves h[depth-1] qubit + branch register at this level
   - For root: the local subspace involves just the branch register (root has no parent transition)
   - Use `from .diffusion import diffusion` and call `diffusion(branch_reg)` for root, or create a temporary wrapper and pass both for non-root.

   IMPORTANT: For the S_0 reflection on non-root nodes, we need to include the h[depth-1] qubit. Since ql.diffusion() takes registers, we need to pass both the branch register AND a qbool wrapping h[depth-1]. Create: `h_child = qbool(create_new=False, bit_list=[self._height_qubit(depth - 1)])` and call `diffusion(h_child, branch_reg)`.
   For root: just `diffusion(branch_reg)`.

   **Step C: U (state preparation, forward)**
   Reverse of U_dagger:
   - For non-root: emit_ry(self._height_qubit(depth - 1), phi), then cascade forward
   - For root: emit_ry on first branch qubit with root_phi (no height transition), then cascade forward

**_ry_cascade(self, branch_reg, d, cascade_angles):**
Apply cascade Ry rotations on branch register to create d-way equal superposition.
- w = branch_reg.width
- If d <= 1: return (no rotation needed)
- If d == 2: single Ry(cascade_angles[0]) on first qubit of branch register (qubit at index 64 - w)
- For d > 2: sequential conditional rotations. For k=0: unconditional Ry(cascade_angles[0]) on qubit 0 (64-w). For k=1..d-2: Ry(cascade_angles[k]) controlled on previous qubit states. Use qbool wrappers on branch register qubits for control.

Note: For the general d > 2 case, the cascade needs careful implementation. The k-th rotation is on a specific branch qubit, conditioned on the state created by previous rotations. For k < w (where w = branch register width), this follows a binary decomposition pattern. The key insight: we need to encode states 0..d-1 with equal amplitude. For d that is a power of 2, this is a balanced Hadamard tree using Ry(pi/2) = H equivalent. For non-power-of-2, the cascade angles theta_k ensure correct amplitude partitioning.

Simplified approach for arbitrary d on w = ceil(log2(d)) qubits:
- Work from MSB to LSB of the branch register
- At each level, split the remaining states into two groups and apply Ry to partition amplitude
- Use controlled rotations when the split depends on a higher bit's state

**_ry_cascade_inverse(self, branch_reg, d, cascade_angles):**
Same as _ry_cascade but with negated angles and reversed order.
