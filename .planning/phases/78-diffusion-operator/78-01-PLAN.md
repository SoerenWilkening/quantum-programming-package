Implement a helper function `_collect_qubits(*registers)` that:
- Iterates over each register argument
- For `qarray`: iterates over elements and collects qubit indices from each element
- For `qint`/`qbool`: collects qubit indices directly
- Returns a flat list of qubit indices
- Qubit indices for a qint of width `w` are at positions `qubits[64 - w + i]` for `i in range(w)` (right-aligned storage pattern from qint.pyx)

Implement a helper `_total_width(*args)` that sums up the total qubit count for the cache key:
- For `qint`/`qbool`: use `.width` property
- For `qarray`: use `len(arr) * arr.width` (total qubits)
- Returns integer total width

Implement the main `diffusion` function:

```python
@ql_compile(key=_total_width)
def diffusion(*registers):
    """Apply Grover diffusion operator (X-MCZ-X pattern).

    Reflects amplitude about |0...0> state. Zero ancilla, O(n) gates.
    Works on qint, qbool, and qarray. Multiple registers are flattened.

    Parameters
    ----------
    *registers : qint, qbool, or qarray
        Quantum registers to apply diffusion to.

    Raises
    ------
    ValueError
        If no qubits provided (empty registers or zero-width).
    """
    qubits = _collect_qubits(*registers)

    if not qubits:
        raise ValueError("diffusion() requires at least one qubit")

    # X on all qubits
    for q in qubits:
        emit_x(q)

    # MCZ: phase flip on |1...1> (which corresponds to |0...0> after X mapping)
    if len(qubits) == 1:
        emit_z(qubits[0])
    else:
        emit_mcz(qubits[-1], qubits[:-1])

    # X on all qubits (undo the mapping)
    for q in qubits:
        emit_x(q)
```

**IMPORTANT:** The `@ql_compile(key=_total_width)` decorator wraps `diffusion` in a `CompiledFunc`. The `key` function receives the same positional args as the function itself. `_total_width` takes `*args` variadic quantum registers and returns an int.
