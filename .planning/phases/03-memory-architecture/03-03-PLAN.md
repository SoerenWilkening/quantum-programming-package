  <action>
Update the qint and qbool classes to allocate qubits through the circuit's allocator.

**Changes to qint class:**

1. In __init__, replace manual qubit tracking with allocator calls:
   ```cython
   # OLD:
   # self.qubits[INTEGERSIZE - bits + i] = _smallest_allocated_qubit + i
   # _smallest_allocated_qubit += bits

   # NEW:
   if create_new:
       # Allocate qubits through circuit's allocator
       cdef unsigned int start = allocator_alloc(_circuit.allocator, bits)
       if start == <unsigned int>-1:
           raise MemoryError("Qubit allocation failed - limit exceeded")

       for i in range(bits):
           self.qubits[INTEGERSIZE - bits + i] = start + i

       self.allocated_start = start  # Track for deallocation
       self.allocated_qubits = True
   ```

2. Add new field to track allocation start:
   ```cython
   cdef unsigned int allocated_start  # Starting qubit index from allocator
   ```

3. In __del__, return qubits to allocator:
   ```cython
   # OLD:
   # _smallest_allocated_qubit -= self.bits

   # NEW:
   if self.allocated_qubits:
       allocator_free(_circuit.allocator, self.allocated_start, self.bits)
   ```

4. Remove or deprecate manual tracking variables:
   - Remove reliance on `_smallest_allocated_qubit` global
   - Remove manual `ancilla` numpy array tracking
   - Keep variables for backward compat but mark as deprecated with comments

**Add circuit statistics accessor:**
```cython
def circuit_stats():
    """Get qubit allocation statistics for the current circuit."""
    if not _circuit_initialized:
        return None
    cdef allocator_stats_t stats = allocator_get_stats(_circuit.allocator)
    return {
        'peak_allocated': stats.peak_allocated,
        'total_allocations': stats.total_allocations,
        'total_deallocations': stats.total_deallocations,
        'current_in_use': stats.current_in_use
    }
```
