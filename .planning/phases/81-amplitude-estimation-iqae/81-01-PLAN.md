Create `src/quantum_language/amplitude_estimation.py` with the following components:

**1. AmplitudeEstimationResult class** (per CONTEXT.md locked decisions):
- Properties: `.estimate` (float), `.num_oracle_calls` (int)
- Optional `.confidence_interval` tuple for internal use
- Float-like behavior: `__float__`, `__add__`, `__radd__`, `__sub__`, `__rsub__`, `__mul__`, `__rmul__`, `__truediv__`, `__rtruediv__`, `__neg__`, `__abs__`, `__eq__`, `__lt__`, `__le__`, `__gt__`, `__ge__`, `__int__`, `__round__`, `__bool__`
- Default `__repr__` (no custom pretty-printing per CONTEXT.md)

**2. Internal helpers:**
- `_simulate_multi_shot(qasm_str, shots)`: Extends the existing `_simulate_single_shot` pattern from `grover.py`. Uses `AerSimulator(max_parallel_threads=4)` (mandatory project constraint). Returns Qiskit counts dict.
- `_count_good_states(counts, register_widths, predicate)`: Parses each bitstring using `_parse_bitstring` from `grover.py`, evaluates classical `predicate` via `_verify_classically` from `grover.py`, returns `(good_counts, total_shots)`.
- `_clopper_pearson_confint(counts, shots, alpha)`: Uses `scipy.stats.beta.ppf` for Clopper-Pearson confidence interval. Handles edge cases (counts=0 returns lower=0, counts=shots returns upper=1).
- `_find_next_k(k, upper_half_circle, theta_interval, min_ratio=2.0)`: FindNextK from Grinko et al. Algorithm 2. Finds largest k_next such that scaled interval stays in one half-circle. Returns `(next_k, upper_half_circle)`.
- `_update_theta_interval(theta_interval, a_min, a_max, k, upper_half_circle)`: Maps measured probability bounds through arccos to theta space, accounts for scaling factor K=4k+2, intersects with previous interval.
- `_build_and_simulate(oracle, register_widths, k, shots)`: Builds fresh circuit with k Grover iterations (reusing pattern from `_run_grover_attempt` in grover.py -- `circuit()`, `option("fault_tolerant", True)`, allocate registers, `branch(0.5)`, k iterations of oracle+H-diffusion-H, `to_openqasm()`), then calls `_simulate_multi_shot`. Returns counts dict.

**3. IQAE main loop `_iqae_loop`:**
- Parameters: `oracle, register_widths, epsilon, alpha, max_iterations, predicate`
- Initial theta interval: `[0.0, 0.25]` (theta/2pi in [0, 1/4])
- Compute `max_rounds = ceil(log(min_ratio * pi / 8 / epsilon) / log(min_ratio)) + 1` with `min_ratio=2.0`
- Compute `shots_per_round = ceil(32 / (1 - 2*sin(pi/14))^2 * ln(2 * max_rounds / alpha))` (from IQAE paper)
- Loop while `theta_interval[1] - theta_interval[0] > epsilon / pi`:
  - Call `_find_next_k` to get next power
  - Call `_build_and_simulate` with k iterations and shots_per_round
  - Call `_count_good_states` to classify measurement outcomes
  - Track `num_oracle_queries += shots_per_round * k`
  - Check `max_iterations` cap on num_oracle_queries; if exceeded, `warnings.warn()` and break
  - Call `_clopper_pearson_confint` with `alpha / max_rounds` (Bonferroni correction)
  - Call `_update_theta_interval` to refine bounds
- Final estimate: `a = (sin(2*pi*theta_l)^2 + sin(2*pi*theta_u)^2) / 2`
- Return `(estimate, num_oracle_queries, confidence_interval)`

**4. Public API `amplitude_estimate(oracle, *registers, width=None, widths=None, epsilon=0.01, confidence_level=0.95, max_iterations=None)`:**
- Mirrors `grover()` signature pattern (per CONTEXT.md locked decision)
- Detect predicate vs decorated oracle (same logic as `grover()`)
- Resolve register widths from `*registers` or `width`/`widths` kwargs (reuse `_resolve_widths` pattern)
- Synthesize oracle from predicate via `_predicate_to_oracle` if needed
- For decorated oracles without predicate: require `predicate` kwarg or raise ValueError explaining that IQAE needs a way to classify measurement outcomes
- Compute `alpha = 1 - confidence_level`
- Warn (but allow) if epsilon is unreasonably small (e.g., `epsilon < 0.001` and total qubits suggest more than ~1000 oracle calls needed)
- Warn when register width sum > 14 (approaching 17-qubit limit with ancilla)
- Call `_iqae_loop`
- Return `AmplitudeEstimationResult(estimate, num_oracle_calls, confidence_interval)`

**Import strategy:**

**Default epsilon=0.01** per Claude's discretion (research recommendation: common IQAE default in literature and Qiskit examples).
