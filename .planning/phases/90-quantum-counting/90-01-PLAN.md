1. **CountResult class** — int-like result wrapper (mirrors AmplitudeEstimationResult pattern from amplitude_estimation.py):
   - Constructor takes `(amplitude_result, search_space)` where amplitude_result is an AmplitudeEstimationResult and search_space is N (integer)
   - `.count` = `max(0, min(N, round(N * amplitude_result.estimate)))` — rounded integer clamped to [0, N]
   - `.estimate` = `N * amplitude_result.estimate` — raw unrounded float
   - `.count_interval` = `(max(0, math.floor(N * ci_low)), min(N, math.ceil(N * ci_high)))` from amplitude confidence interval. If confidence_interval is None, use (count, count)
   - `.search_space` = N as integer
   - `.num_oracle_calls` = amplitude_result.num_oracle_calls
   - Int-like behavior: `__int__` returns count, `__float__` returns estimate, `__bool__` returns bool(count)
   - Arithmetic operators: `__add__`, `__radd__`, `__sub__`, `__rsub__`, `__mul__`, `__rmul__`, `__truediv__`, `__rtruediv__`, `__neg__`, `__abs__` — all operate on `self._count`
   - Comparison operators: `__eq__`, `__lt__`, `__le__`, `__gt__`, `__ge__` — compare self._count to int(other). For `__eq__`, handle TypeError/ValueError returning NotImplemented
   - `__repr__` returning `CountResult(count={count}, estimate={estimate:.4f}, search_space={N})`
   - `__round__` returning round(self._count, ndigits) (for consistency)
   - Zero is a valid count — no special case or sentinel
