    **Step 1: Add validation helper functions in `_core.pyx`**

    Add two inline validation helper functions near the top of `_core.pyx` (after the module-level globals):

    ```python
    cdef inline void _validate_circuit() except *:
        """Raise ValueError if circuit pointer is NULL or uninitialized."""
        if not _circuit_initialized or _circuit == NULL:
            raise ValueError(
                "[Validation] error in circuit operation: circuit pointer is NULL"
            )

    cdef inline void _validate_qubit_slots(int required, str func_name) except *:
        """Raise OverflowError if required qubit_array slots exceed 384."""
        if required > 384:
            raise OverflowError(
                f"[Buffer] error in {func_name}: slot count exceeded, max 384"
            )
    ```

    Also export `_validate_circuit` and `_validate_qubit_slots` as Python-callable functions for use from other .pyx files:

    ```python
    def validate_circuit():
        """Public wrapper for circuit pointer validation."""
        _validate_circuit()

    def validate_qubit_slots(int required, str func_name):
        """Public wrapper for qubit_array bounds validation."""
        _validate_qubit_slots(required, func_name)
    ```

    **Step 3: Add qubit_array bounds pre-checks in qint operations**

    In `qint.pyx` (and the `.pxi` include files it uses), add `_validate_qubit_slots()` calls before blocks that populate `qubit_array`. The key operations are:

    For each arithmetic operation in `qint_arithmetic.pxi` that writes to `qubit_array`:
    - `addition_inplace` / `__iadd__`: Calculate total slots needed (depends on operand widths + ancilla) and call `_validate_qubit_slots(total, "addition_inplace")` before writing
    - `multiplication` / `__mul__`: Calculate total slots and validate before writing
    - Similar for subtraction, division operations in `qint_division.pxi`

    For bitwise operations in `qint_bitwise.pxi`:
    - `__ixor__`, `__iand__`, `__ior__`: Calculate slots and validate

    For comparison operations in `qint_comparison.pxi`:
    - `__lt__`, `__gt__`, `__le__`, `__ge__`, `__eq__`, `__ne__`: Calculate slots and validate

    The slot count calculation for each operation is: sum of all distinct `qubit_array[index]` assignments in that code path. For most operations this is bounded by `3 * result_bits + ancilla_count` which for 64-bit operands would be ~3*64 + some ancilla = well within 384. The check ensures pathological inputs don't overflow.

    Import the validation functions from `_core`:
