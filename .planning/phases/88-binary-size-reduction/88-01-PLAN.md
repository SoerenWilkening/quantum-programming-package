  <action>
Modify setup.py to support Release and Debug build modes with size optimization flags.

**Build mode detection:**
- Read `BUILD_TYPE` environment variable; default to "Release" when not set
- This means `pip install .` and `python setup.py build_ext --inplace` default to Release (size-optimized)
- Developers use `BUILD_TYPE=Debug` for development builds

**Release mode flags (SIZE-01 + SIZE-02):**
- Replace the current `compiler_args = ["-O3", "-pthread"]` with conditional logic
- Release compiler_args: `["-O3", "-ffunction-sections", "-fdata-sections", "-pthread"]`
  - NOTE: Keep -O3 for now (Plan 02 will evaluate -Os vs -O3)
- Release linker_args: platform-conditional:
  - Linux: `["-Wl,--gc-sections", "-s"]`
  - macOS (sys.platform == "darwin"): `["-Wl,-dead_strip", "-Wl,-x"]`

**Debug mode flags:**
- Debug compiler_args: `["-O3", "-pthread"]` (same as current)
- Debug linker_args: `[]` (same as current)

**Critical: Do NOT change profiling/coverage behavior.**
- The existing `QUANTUM_PROFILE` and `QUANTUM_COVERAGE` environment variable checks must still work
- Profiling/coverage flags append to compiler_args and linker_args as before
- If `QUANTUM_PROFILE` or `QUANTUM_COVERAGE` is set, override to Debug-like flags (no stripping, no gc-sections) because coverage instrumentation and stripping are incompatible

**Implementation pattern:**
```python
import sys

# Build type: Release by default (optimized for size), Debug via BUILD_TYPE=Debug
build_type = os.environ.get("BUILD_TYPE", "Release")
is_release = build_type.lower() == "release"

# Override: profiling/coverage modes need debug-like flags
if os.environ.get("QUANTUM_PROFILE") or os.environ.get("QUANTUM_COVERAGE"):
    is_release = False

# Base compiler and linker flags
if is_release:
    compiler_args = ["-O3", "-ffunction-sections", "-fdata-sections", "-pthread"]
    if sys.platform == "darwin":
        linker_args = ["-Wl,-dead_strip", "-Wl,-x"]
    else:
        linker_args = ["-Wl,--gc-sections", "-s"]
else:
    compiler_args = ["-O3", "-pthread"]
    linker_args = []
```

Add `import sys` at the top if not already imported.
