  <name>Task 1: Implement CQ_equal_width function</name>
  <files>Backend/src/IntegerComparison.c, Backend/include/comparison_ops.h</files>
  <action>
Replace the stub CQ_equal_width implementation with a working gate sequence generator.

Algorithm (XOR-based, O(n) gates):
1. For each bit position i:
   - If classical_bit[i] == 1: do nothing (quantum bit should be 1)
   - If classical_bit[i] == 0: apply X gate (flip quantum bit, so it should become 0)
2. After step 1: quantum register equals classical value IFF all qubits are |0>
3. Use multi-controlled X (Toffoli cascade) to set result qubit to |1> if all operand qubits are |0>
4. Reverse the X gates from step 1 to restore original quantum state

Edge cases:
- If value >= 2^bits (value cannot fit): return empty sequence with 0 gates (comparison always false)
- If bits <= 0 or bits > 64: return NULL
- If value < 0: use two's complement representation

Qubit layout:
- [0]: Result qubit (output, set to |1> if equal)
- [1, bits]: Quantum operand to compare

Memory management:
- Follow existing pattern from CQ_and/CQ_or in LogicOperations.c
- Caller owns returned sequence (must free)
- Use two_complement() from Integer.h for binary conversion

The function signature stays the same: sequence_t *CQ_equal_width(int bits, int64_t value)
