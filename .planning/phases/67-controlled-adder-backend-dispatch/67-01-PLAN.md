  <action>
Add controlled MAJ/UMA helpers and controlled adder functions to ToffoliAddition.c:

**1. Add emit_cMAJ and emit_cUMA static helpers** (after existing emit_MAJ/emit_UMA):

`emit_cMAJ(seq, layer, a, b, c, ext_ctrl)`:
- Step 1: CCX(target=b, ctrl1=c, ctrl2=ext_ctrl) -- controlled b ^= c
- Step 2: CCX(target=a, ctrl1=c, ctrl2=ext_ctrl) -- controlled a ^= c
- Step 3: MCX(target=c, controls=[a, b, ext_ctrl]) -- controlled c ^= (a AND b), 3 controls
- Each gate on its own layer (same pattern as emit_MAJ)

`emit_cUMA(seq, layer, a, b, c, ext_ctrl)`:
- Step 1: MCX(target=c, controls=[a, b, ext_ctrl]) -- 3 controls
- Step 2: CCX(target=a, ctrl1=c, ctrl2=ext_ctrl)
- Step 3: CCX(target=b, ctrl1=a, ctrl2=ext_ctrl)
- Each gate on its own layer

For MCX calls: declare `qubit_t ctrls[3] = {a, b, ext_ctrl};` and call `mcx(&seq->seq[*layer][seq->gates_per_layer[*layer]++], target, ctrls, 3);`

**2. Add `toffoli_cQQ_add(int bits)` public function**:

Cache array: `static sequence_t *precompiled_toffoli_cQQ_add[65] = {NULL};`

Qubit layout:
- `[0..bits-1]` = register a (target, gets sum)
- `[bits..2*bits-1]` = register b (source, preserved)
- `[2*bits]` = ancilla carry (bits >= 2 only)
- `[2*bits+1]` = external control qubit

1-bit special case (bits == 1): No ancilla. Layout: [0]=a, [1]=b, [2]=ext_control. Single CCX(target=0, ctrl1=1, ctrl2=2). Total qubits: 3.

General case (bits >= 2): `6 * bits` layers. `ext_ctrl = 2 * bits + 1`. Forward cMAJ sweep + reverse cUMA sweep, same structure as toffoli_QQ_add but using emit_cMAJ/emit_cUMA with ext_ctrl parameter. Cache the result.

OWNERSHIP: Returns cached sequence - DO NOT FREE.

**3. Add `toffoli_cCQ_add(int bits, int64_t value)` public function**:

Qubit layout:
- `[0..bits-1]` = temp register (controlled init to classical value, controlled cleanup)
- `[bits..2*bits-1]` = self register (target, gets sum)
- `[2*bits]` = carry ancilla (bits >= 2 only)
- `[2*bits+1]` = external control qubit

1-bit special case (bits == 1): If value's LSB == 1, emit CX(target=0, ctrl=1) where [0]=self, [1]=ext_control. If LSB == 0, return empty sequence. Total qubits: 2.

General case (bits >= 2):
- Phase 1: Controlled X-init temp register. For each bit i (LSB-first), if classical bit is 1, emit CX(target=i, control=ext_ctrl) where ext_ctrl = 2*bits+1. This ensures temp is only initialized when control is |1>.
- Phase 2: Controlled QQ CDKM adder. Use the SAME emit_cMAJ/emit_cUMA chain as cQQ, operating on temp (a-register) and self (b-register) with ext_ctrl.
- Phase 3: Controlled X-cleanup temp register. Same CX gates as Phase 1 (CX is self-inverse when temp has been preserved by CDKM).

Total layers: `2 * x_count + 6 * bits` (same structure as uncontrolled CQ but with CX instead of X for init/cleanup, and cMAJ/cUMA instead of MAJ/UMA).

OWNERSHIP: Caller must free via toffoli_sequence_free(). NOT cached (value-dependent).

**4. Fix toffoli_sequence_free() for large_control cleanup**:

Before freeing each layer's gate array `seq->seq[i]`, iterate over gates in that layer (up to `seq->gates_per_layer[i]`) and free `gate.large_control` if `gate.NumControls > 2 && gate.large_control != NULL`. This prevents memory leaks from MCX gates with 3+ controls.

**5. Update toffoli_arithmetic_ops.h**:

Add declarations:
```c
sequence_t *toffoli_cQQ_add(int bits);
sequence_t *toffoli_cCQ_add(int bits, int64_t value);
```

With documentation blocks specifying qubit layouts and ownership contracts.

IMPORTANT ANTI-PATTERNS TO AVOID:
- Do NOT put control qubit at index 2*bits for cQQ (collides with ancilla). Control goes at 2*bits+1.
- Do NOT use unconditional X gates for cCQ init/cleanup. Use CX(target=temp[i], control=ext_ctrl).
- Do NOT cache cCQ sequences (they are value-dependent).
