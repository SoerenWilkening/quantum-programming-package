**Root cause:** `_PhaseProxy.__iadd__` already checks `_get_controlled()` and gets the control qubit. It then calls `emit_p(ctrl.qubits[63], theta)`. But `emit_p` ALSO checks `_get_controlled()` and wraps the call with CP, producing `cp(target=ctrl.qubits[63], control=ctrl.qubits[63], angle)` -- the same qubit is both target and control. This self-controlled gate is just `P(angle)` on the ancilla qubit, which has zero effect on the search register. When the comparison qbool is uncomputed, the ancilla returns to |0> and the phase is lost.

**Fix strategy:** Add a new `emit_p_raw` function to `_gates.pyx` that emits a P gate WITHOUT checking the controlled context. Then update `_PhaseProxy.__iadd__` to call `emit_p_raw` instead of `emit_p`.

```python
cpdef void emit_p_raw(unsigned int target, double angle):
    """Emit P(angle) gate to circuit WITHOUT auto-control context.

    Unlike emit_p, this does NOT check _get_controlled() and always
    emits a plain P gate. Used by _PhaseProxy.__iadd__ which handles
    the controlled context itself -- calling emit_p from inside a
    controlled context would double-apply the control, producing
    cp(q[n], q[n]) instead of p(q[n]).

    Parameters
    ----------
    target : unsigned int
        Target qubit index for P gate
    angle : double
        Phase angle in radians
    """
    cdef gate_t g
    cdef circuit_t *circ = <circuit_t*><unsigned long long>_get_circuit()

    memset(&g, 0, sizeof(gate_t))
    p(&g, target, angle)
    add_gate(circ, &g)
```

```python
    if _get_controlled():
        ctrl = _get_control_bool()
        # Use emit_p_raw to avoid double-control: emit_p would check
        # _get_controlled() again and wrap with CP, producing
        # cp(ctrl, ctrl) -- a self-controlled gate with no search
        # register effect. Instead, emit P directly on the control
        # qubit. P(theta) on a qubit in |1> adds phase e^{i*theta}
        # to all basis states where that qubit is |1>, which is
        # exactly the controlled global phase we want.
        emit_p_raw(ctrl.qubits[63], theta)
    # Uncontrolled: no gate (global phase is unobservable)
    return self
```

**Why this works:** When `with x == 0:` is active, the comparison ancilla qubit is |1> for basis states where x == 0 and |0> otherwise. Applying `P(pi)` directly to the ancilla (without auto-control wrapping) adds a phase of `e^{i*pi} = -1` to all basis states where the ancilla is |1>, i.e., where x == 0. When the comparison is uncomputed, the ancilla is disentangled but the phase remains imprinted on the search register via phase kickback. The net effect is the S_0 reflection: phase flip on |0...0>.

**IMPORTANT:** The `emit_p_raw` approach is correct because `P(theta)` on the control qubit in a controlled context implements a controlled global phase via phase kickback. The key insight is that the controlled context machinery has ALREADY set up the ancilla qubit to encode the condition -- we just need to apply P to it, not CP.

**Note on QASM output:** After this fix, the QASM will show `p(3.14159...) q[n]` (a plain P gate on the ancilla) instead of `cp(3.14159...) q[n], q[n]`. The P gate will appear BETWEEN the comparison gates and their reverse (uncomputation). Both the P gate and the uncomputation survive -- the P gate adds the phase while the ancilla is |1>, and the uncomputation then disentangles the ancilla.

The new test should:
- Create a fresh circuit with `ql.circuit()` and `ql.option('fault_tolerant', True)`
- Create a 2-qubit register `x = ql.qint(0, width=2)`
- Apply `x.branch()` to create equal superposition
- Apply manual S_0: `with x == 0: x.phase += math.pi`
- Export QASM via `ql.to_openqasm()`
- Simulate statevector via `_simulate_statevector(qasm)`
- Verify that the |00> amplitude has opposite sign from the other amplitudes

**IMPORTANT Qiskit simulation note:** The QASM will have more qubits than just the 2-qubit search register because the comparison `x == 0` allocates ancilla qubit(s). The statevector will have 2^(total_qubits) entries. To check the search register state, we need to trace out (marginalize) the ancilla qubits or check the correct indices. Since the ancilla is uncomputed back to |0>, the statevector factorizes as |ancilla=0> tensor |search_state>. The search register amplitudes are in the entries where all ancilla qubits are 0.

Approach for extracting search register state from statevector with ancilla:
- Count total qubits from QASM `qubit[N]` declaration
- Search register is the first 2 qubits (indices 0, 1 in Qiskit's little-endian convention)
- Ancilla qubits are the remaining qubits (indices 2..N-1)
- For entries where ancilla bits are all 0, extract the search register amplitudes
- With Qiskit's little-endian convention: index `i` in the statevector has qubit 0 = bit 0 of `i`, qubit 1 = bit 1, etc. So ancilla qubits correspond to higher bits.
- Entries where ancilla bits are 0: indices 0 to 2^(search_width)-1

So for a 2-qubit search register with N total qubits:
- Indices 0, 1, 2, 3 correspond to search states |00>, |01>, |10>, |11> with ancilla = |0...0>
- These are the first 4 entries of the statevector

```python
def test_manual_s0_reflection_statevector(self):
    """Manual S_0: with x == 0: x.phase += pi produces correct S_0 reflection.

    Directly tests the manual path via Qiskit statevector simulation.
    After branch() + manual S_0, the |00> amplitude should have opposite
    sign from the other amplitudes (same effect as ql.diffusion()).
    """
    ql.circuit()
    ql.option("fault_tolerant", True)

    x = ql.qint(0, width=2)
    x.branch()
    with x == 0:
        x.phase += math.pi

    qasm = ql.to_openqasm()
    sv = _simulate_statevector(qasm)

    # Extract search register amplitudes (first 2^width entries,
    # where all ancilla qubits are in |0> state)
    search_width = 2
    n_search_states = 2 ** search_width
    amps = [complex(sv[i]).real for i in range(n_search_states)]

    # After branch() (equal superposition) + S_0 reflection:
    # |00> amplitude should have opposite sign from others
    amp_00 = amps[0]
    amp_others = amps[1:]

    assert amp_00 < 0, (
        f"|00> should be negative after S_0 reflection: {amp_00:.6f}\n"
        f"All amps: {amps}\nQASM:\n{qasm}"
    )
    assert all(a > 0 for a in amp_others), (
        f"Non-|00> amps should be positive: {amp_others}\n"
        f"All amps: {amps}\nQASM:\n{qasm}"
    )

    # Verify magnitudes are equal (uniform was input)
    assert np.isclose(abs(amp_00), abs(amp_others[0]), atol=1e-6), (
        f"Amplitudes should have equal magnitude: "
        f"|00|={abs(amp_00):.6f}, |01|={abs(amp_others[0]):.6f}"
    )
```
