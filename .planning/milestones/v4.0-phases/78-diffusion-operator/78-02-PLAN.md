**Helper functions (module-level):**

```python
def _simulate_statevector(qasm_str):
    """Run QASM through Qiskit Aer and return statevector."""
    from qiskit import qasm3, transpile
    from qiskit_aer import AerSimulator
    circuit = qasm3.loads(qasm_str)
    circuit.save_statevector()
    sim = AerSimulator(method='statevector')
    result = sim.run(transpile(circuit, sim)).result()
    return result.get_statevector()

def _simulate_counts(qasm_str, shots=8192):
    """Run QASM through Qiskit Aer and return measurement counts."""
    from qiskit import qasm3, transpile
    from qiskit_aer import AerSimulator
    circuit = qasm3.loads(qasm_str)
    circuit.measure_all()
    sim = AerSimulator()
    result = sim.run(transpile(circuit, sim), shots=shots).result()
    return result.get_counts()
```

3. `test_diffusion_3qubit_qasm` -- Apply diffusion to 3-qubit register. Verify QASM output contains X gates and multi-controlled Z. Count X gates (should be 6 = 3 before + 3 after).

4. `test_diffusion_statevector_2qubit` -- Create 2-qubit register, apply `branch()` (equal superposition), then `diffusion()`. Export QASM, simulate with Qiskit statevector. Key check: the |00> amplitude should have the opposite sign from the others (since S_0 flips |0...0>).

5. `test_diffusion_statevector_3qubit` -- Same as above for 3 qubits. After `branch()` (equal superposition of 8 states), `diffusion()` should reflect about |0...0>. Verify: amplitude of |000> is different sign from the other 7 basis states.

6. `test_diffusion_multi_register` -- Apply `ql.diffusion(x, y)` where x is 2-qubit and y is 1-qubit. Total width is 3 qubits. Verify QASM output is equivalent to diffusion on a single 3-qubit register (6 X gates + MCZ).

7. `test_diffusion_qbool` -- Apply `ql.diffusion(b)` where b is a qbool. Verify it works (1-qubit diffusion = Z sandwiched by X).

8. `test_diffusion_qarray` -- Apply `ql.diffusion(arr)` where arr is a qarray of 2 qbool elements. Verify total width is 2 qubits and QASM shows 4 X gates + CZ.

9. `test_diffusion_zero_width_error` -- Verify `ql.diffusion()` with no arguments raises ValueError.

10. `test_diffusion_compile_caching` -- Call `ql.diffusion(x)` twice on same-width register. Verify second call hits the compile cache (doesn't re-capture). Can check by comparing QASM output is identical or by checking diffusion._cache or similar.

11. `test_diffusion_controlled` -- Apply `ql.diffusion(x)` inside a `with flag:` block where flag is a qbool. Verify QASM shows controlled gates (CX instead of X, controlled-MCZ). This tests the @ql.compile controlled derivation.
