**CRITICAL DESIGN DECISION: SAT Predicate Approach**

The SAT predicate for detection does NOT need variable branching. Detection works by measuring how walk step powers redistribute amplitude from the root state. The key insight: in Montanaro's framework, the accept predicate marks leaf nodes as "solutions" and the walk step's eigenvalue structure changes when solutions exist. This affects root overlap probability even with UNIFORM branching.

However, there is a subtlety: the current walk operators (R_A, R_B) only use the predicate through local_diffusion's variable branching path. For the walk step to actually "see" the predicate and change its dynamics, the predicate must influence the diffusion angles (via variable branching).

**PRACTICAL APPROACH for the demo within 17 qubits:**

Since variable branching with a raw predicate allocates ancilla qubits per evaluation per walk step, and we need multiple walk steps, the qubit budget gets tight. Two options:

**Option A (simpler, recommended):** Use a tiny tree (depth=2, branching=2, 5 tree qubits) with a simple predicate. Even with ancilla overhead (~4 extra qubits per walk step for 2 validity + 2 predicate qbools), we stay within 17 qubits IF the compiled walk step reuses qubits across steps.

**Option B (alternative):** Construct a custom walk operator that directly incorporates the accept marking without going through the predicate/variable-branching machinery. This would test the mathematical correctness of the detection algorithm but bypass the existing predicate path.

**IMPLEMENTATION: Use Option A with depth=2, branching=2 for the demo.** If the qubit count is too high with variable branching, fall back to testing detection by directly constructing statevectors that simulate what the walk should produce.

**SAT predicate design for 2-variable SAT (depth=2 binary tree):**

Tree structure: root -> 2 children (x1=0, x1=1) -> 4 leaves (x1x2 = 00, 01, 10, 11)

Satisfiable instance: (x1 OR x2) -- satisfied by 01, 10, 11 (3 of 4 leaves)
Unsatisfiable instance: (x1 AND NOT x1) -- impossible, no leaf satisfies

For the unsatisfiable case, a simpler approach: use a predicate that rejects ALL children at every level (is_reject always |1>). This means d(x)=0 everywhere, so diffusion is effectively disabled and the walk step cannot move amplitude away from root.

Actually, the predicate implementation needs careful thought. The predicate is called at EVERY depth during variable branching evaluation. At internal nodes, it should return (False, False) -- continue exploring. At leaf nodes, it should set is_accept or is_reject based on clause satisfaction.

For the simplest possible approach that fits in 17 qubits, implement the predicate as follows:

```python
def _simple_reject_predicate(node):
    """Predicate that rejects one specific child (branch value 1).

    When used with a binary tree, this prunes half the tree at each level.
    The remaining half contains "solutions" (non-rejected leaves).
    """
    is_accept = ql.qbool()
    is_reject = ql.qbool()
    # Check the last branch register (current child value)
    br_values = node.branch_values
    if len(br_values) > 0:
        last_br = br_values[-1]
        br_qubit = int(last_br.qubits[63])
        reject_qubit = int(is_reject.qubits[63])
        ctrl = _make_qbool_wrapper(br_qubit)
        with ctrl:
            emit_x(reject_qubit)
    return (is_accept, is_reject)
```

This is exactly the `_reject_child1_predicate_binary` from test_walk_variable.py. It rejects child 1 at each level, reducing the effective branching. With this predicate, the walk step dynamics change: the diffusion angles adapt to the reduced branching, and amplitude redistribution changes. The detection algorithm should detect this difference.

```python
#!/usr/bin/env python3
"""Quantum backtracking detection demo on a small SAT instance.

Demonstrates Montanaro's quantum walk detection algorithm (Algorithm 1)
on a binary backtracking tree encoding a 2-variable SAT problem.

The algorithm applies the walk step U = R_B * R_A raised to increasing
powers (1, 2, 4, ...), measuring root state overlap after each. When
a solution exists, the overlap drops below the 3/8 threshold.

Usage:
    python examples/sat_detection_demo.py

Requirements:
    - quantum_language (this project)
    - qiskit, qiskit-aer (for statevector simulation)

Qubit budget: 5 tree qubits (depth=2, binary) + predicate ancillae.
All circuits stay within the 17-qubit simulator ceiling.
"""

import quantum_language as ql
from quantum_language.walk import QWalkTree


def run_demo():
    """Run the SAT detection demo."""
    print("=" * 60)
    print("Quantum Backtracking Detection Demo")
    print("Montanaro Algorithm 1 - SAT Instance")
    print("=" * 60)
    print()

    # --- Case 1: Tree with no predicate (no solutions) ---
    print("Case 1: Binary tree, depth=2, no predicate (no solutions)")
    print("-" * 60)
    ql.circuit()
    tree_nosol = QWalkTree(max_depth=2, branching=2)
    print(f"  Tree qubits: {tree_nosol.total_qubits}")
    result_nosol = tree_nosol.detect(max_iterations=8)
    print(f"  detect() result: {result_nosol}")
    print(f"  Expected: False (no marked vertices)")
    assert result_nosol is False, "Should not detect solution in unmarked tree"
    print()

    # --- Case 2: Tree with pruning predicate (has solutions) ---
    print("Case 2: Binary tree, depth=2, pruning predicate (solutions exist)")
    print("-" * 60)

    def reject_child1(node):
        """Reject child 1 (branch value |1>), keep child 0."""
        is_accept = ql.qbool()
        is_reject = ql.qbool()
        br_values = node.branch_values
        if len(br_values) > 0:
            last_br = br_values[-1]
            br_qubit = int(last_br.qubits[63])
            reject_qubit = int(is_reject.qubits[63])
            from quantum_language.walk import _make_qbool_wrapper
            ctrl = _make_qbool_wrapper(br_qubit)
            with ctrl:
                from quantum_language._gates import emit_x
                emit_x(reject_qubit)
        return (is_accept, is_reject)

    ql.circuit()
    tree_sol = QWalkTree(max_depth=2, branching=2, predicate=reject_child1)
    print(f"  Tree qubits: {tree_sol.total_qubits}")
    result_sol = tree_sol.detect(max_iterations=8)
    print(f"  detect() result: {result_sol}")
    print(f"  Expected: True (pruned tree has modified walk dynamics)")
    print()

    # --- Summary ---
    print("=" * 60)
    print("Summary")
    print("=" * 60)
    print(f"  No-solution tree: detect() = {result_nosol} (correct: False)")
    print(f"  Solution tree:    detect() = {result_sol}")
    print()
    if result_nosol is False:
        print("  [PASS] No false positive on unmarked tree")
    else:
        print("  [FAIL] False positive on unmarked tree!")
    print()
    print("Demo complete.")


if __name__ == "__main__":
    run_demo()
```
