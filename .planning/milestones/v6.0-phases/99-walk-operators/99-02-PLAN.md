```python
def _simulate_statevector(qasm_str):
    circuit = qiskit.qasm3.loads(qasm_str)
    circuit.save_statevector()
    sim = AerSimulator(method="statevector", max_parallel_threads=4)
    result = sim.run(transpile(circuit, sim)).result()
    return np.asarray(result.get_statevector())

def _qubit_state_index(*qubit_indices):
    idx = 0
    for q in qubit_indices:
        idx |= 1 << q
    return idx
```

Mathematical note -- walk step U = R_B * R_A is NOT a reflection (U^2 != I in general).
U is a unitary whose eigenvalues are related to the spectrum of the walk operator.
For the primary test (binary tree depth=2, 5 qubits), statevector comparison verifies
that U applied to the root state produces a specific non-trivial superposition.
